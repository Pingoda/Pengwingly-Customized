### This ModPack aims to improve Customizability, Performance and help with Content Creation adding various Tools to the game.

![Content Creation](https://cdn.modrinth.com/data/cached_images/184af64091c7e85ea2b427355481a8e1352c5346.png)

### Pengwingly Customized offers various different Tools for Content Creator that come already preinstalled in the ModPack such as the [Replay Mod](https://www.replaymod.com/) & [Litematica](https://www.curseforge.com/minecraft/mc-mods/litematica).

![Performance](https://cdn.modrinth.com/data/cached_images/a3683d91d070dd5cb929678bbd867a0b81a490ca.png)
### Since Pengwingly Customized is based off [Fabulously Optimized](https://modrinth.com/modpack/fabulously-optimized) it grants all the performance gains from the best Minecraft Performance Mods.

![OptiFine Parity](https://cdn.modrinth.com/data/cached_images/77c34a4b19b513232f857b0acce43f93e0b7362c.png)
### Pengwingly Customized supports almost all (and more) of the features bundled with **OptiFine** without using it.

![Client-Side](https://cdn.modrinth.com/data/cached_images/948cad54b348cd7a2884fbc8fe32d21496b0e725.png)
### Pengwingly Customized is meant to work on any Server without needing the Mods on the Server or risking the ban, but make sure that the Mods in Pengwingly Customized are allowed since different Servers may have different rules!

![Help & Support](https://cdn.modrinth.com/data/cached_images/3ec08f2fbf5d686c0dc03c428022a84c57d4c258.png)
### Pengwingly Customized is meant to be a ModPack for everyone, if you want a specific feature in the ModPack make sure to let me know through [GitHub](https://github.com/Pingoda/Pengwingly-Customized) or Discord (soon™), and I'll see if it's possible!
If you need any help with the ModPack (bear in mind I won't be helping you if you modify the ModPack) make sure to check out the [GitHub Repo](https://github.com/Pingoda/Pengwingly-Customized) or the Discord (soon™) for anything that you might need!
